## 🚀 Getting Started

To run this project locally:

1. **Clone the repository**

```bash
git clone https://github.com/your-username/your-repo-name.git
````

2. **Navigate into the project directory**

```bash
cd your-repo-name
```

3. **Install dependencies**

```bash
npm install
```

4. **Run the development server**

```bash
npm run dev
```
